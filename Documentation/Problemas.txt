Problemas del proyecto
    - Ontología vs Base de Datos
        1: Imposible asignar definición a las clases de la ontología.
            -> Solución: Crear clase abstracta, y crear un individuo por clase, con la definición correspondiente

        2: Calcular requisitos que requieren valores numéricos de las habilidades
            -> Solución: Utilizar data properties en el personaje para indicar dichos valores
